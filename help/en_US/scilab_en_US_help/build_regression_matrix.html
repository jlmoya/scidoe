build_regression_matrix

Build a regression matrix to learn a polynomial

Calling Sequence

R = build_regression_matrix(H,model,build)

Parameters

H

a design of experiment or a vector

model

a list of monomials like '1 x1 x2, x1*x2' where xi corresponds to the ith column

build

a boolean vector of the same size as model. If build(i) == %T then the monomial i will be selected. By default, all the monomials are selected

R

the regression matrix

Description

Build a regression matrix to learn a polynomial

Examples

model_lin   = '1 x1 x2';

x = -1:0.1:1;
y = -1:0.1:1;
[X,Y] = meshgrid(x,y);

////////////////////////////
// Test of classicals DoE //
////////////////////////////

// Test of the factorial DoE on a linear model
H1  = doe_factorial(2);
RM1 = build_regression_matrix(H1,model_lin);
