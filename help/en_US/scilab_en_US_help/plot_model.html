plot_model

plot a sum-up of the caracteristics of a model

Calling Sequence

plot_model(meas_learn,estim_learn,meas_valid,estim_valid)

Parameters

meas_learn

the set of output measured data which have been used to learn the model

estim_learn

the set of output estimated (by the model) data which have been used to learn the model

meas_valid

the set of output measured data which have been used for the validation of the model

estim_valid

the set of output estimated (by the model) data which have been used for the validation of the model

Description

Plot a sum-up of the caracteristics of a model.

Examples

rand('norm');
meas_learn = rand(100,1);
estim_learn = meas_learn + 0.1;

meas_valid = rand(20,1);
estim_valid = meas_valid + 0.2;

plot_model(meas_learn, estim_learn, meas_valid, estim_valid);
