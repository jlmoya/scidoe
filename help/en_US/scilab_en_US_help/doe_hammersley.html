doe_hammersley

Computes a Hammersley quasi-random set of points

Calling Sequence

r = doe_hammersley(dim_num,n,step,seed,leap,base)

Parameters

dim_num

the number of variables for the design of experiments

n

the number of points to generate

step

the index of the subsequence element. 0 <= step is required.

seed

the sequence index corresponding to step = 0 (a vector of size dim_num).

leap

the succesive jumps in the sequence (a vector of size dim_num)

base

the bases (a vector of size dim_num)

r

Add here the parameter description

Description

Computes a Hammersley quasi-random set of points.

Examples

Dimension = 2;
NbPts     = 100;

List_Halton     = doe_halton(Dimension, NbPts);
List_Hammersley = doe_hammersley(Dimension, NbPts);
List_Faure      = doe_faure(Dimension, NbPts);

h = scf();
subplot(2,2,1);
plot(List_Halton(:,1), List_Halton(:,2), 'k.');
xtitle('Halton random sequence','x1','x2');
subplot(2,2,2);
plot(List_Hammersley(:,1), List_Hammersley(:,2), 'k.');
xtitle('Hammersley random sequence','x1','x2');
subplot(2,2,3);
plot(List_Faure(:,1), List_Faure(:,2), 'k.');
xtitle('Faure random sequence','x1','x2');
