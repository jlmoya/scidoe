doe_poly_model

Produce a list of monomials corresponding to a given polynomial

Calling Sequence

model = doe_poly_model(mod_type,nb_var,order)

Parameters

mod_type

the type of polynomial we want to produce. Can be:

  • 'lin': linear model

  • 'poly': pure polynomial model

  • 'inter': linear model with interaction

nb_var

number of variables in the model

order

the order of the polynomial

model

the list of monomials which compose the model

Description

Produce a list of monomials corresponding to a given polynomial

Examples

disp(doe_poly_model('lin',3,2));
disp(doe_poly_model('poly',3,2));
disp(doe_poly_model('inter',3,2));
