comp_o_opti_crit

Computes the value of the O optimal criterion

Calling Sequence

Result = comp_o_opti_crit(M_doe,model)

Parameters

M_doe

a design of experiment or a vector.

model

a list of monomials like '1 x1 x2, x1*x2' where xi corresponds to the ith column.

Result

the value of the G optimal criterion for this model and design of experiments.

Description

Computes the value of the O optimal criterion.
