doe_test_significance

Statistical test based on the student test. This test allows to test the level of significance of a parameter.

Calling Sequence

result = doe_test_significance(param_mean,param_var,size_stat,val_to_comp,level,operation)

Parameters

param_mean

the mean value of the considered parameter

param_var

the variance of the considered parameter

size_stat

the size of the sample which has been used to computed param_mean and param_var

val_to_comp

to which value do we compare

level

the level of confidence of the test

operation

which kind of operation do we want to perform. Can be '==', '>=' or '<='

result

a boolean corresponding to the result of the test.

Description

Statistical test based on the student test. This test allows to test the level of significance of a parameter.

Examples

A_mean = 10.1;
A_var  = 0.5;
size_stat = 27;

disp(doe_test_significance(A_mean, A_var, size_stat, 2, 0.05, '=='));
disp(doe_test_significance(A_mean, A_var, size_stat, 2, 0.05, '>='));
disp(doe_test_significance(A_mean, A_var, size_stat, 2, 0.05, '<='));
