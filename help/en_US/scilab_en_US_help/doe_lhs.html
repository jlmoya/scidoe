doe_lhs

Computes a Latin Hypercube Sampling design of experiments

Calling Sequence

lhs_matrix = doe_lhs(nb_dims,x_min,x_max,nb_div,nb_iterations,nb_points,random)

Parameters

nb_dims

the number of variables for the design of experiments

x_min

the vector of lower bounds for each variables

x_max

the vector of upper bounds for each variables

nb_div

the number of divisions for each axes. Default values: nb_dims

nb_iterations

the number of iterations during which we compute the lhs points. Default value: 3*nb_point

nb_points

the number of points we want in the resulting design of experiments

random

if % T, then the points of the LHS design of experiments will be placed at a random position in the given cell.

lhs_matrix

the optimized design of experiments

Description

Computes a Latin Hypercube Sampling design of experiments.

Examples

nb_dims = 2;
xmin    = -1*ones(1,nb_dims);
xmax    =  1*ones(1,nb_dims);
nb_div  = 10;
nb_points     = 100;
nb_iterations = 3*nb_points;  

lhs_out_1 = doe_lhs(nb_dims,xmin,xmax,nb_div,nb_iterations,nb_points,%F);
lhs_out_2 = doe_lhs(nb_dims,xmin,xmax,nb_div,nb_iterations,nb_points,%T);

subplot(1,2,1);
plot(lhs_out_1(:,1), lhs_out_1(:,2), 'r.');
xtitle('LHS deterministic','x1','x2');

subplot(1,2,2);
plot(lhs_out_2(:,1), lhs_out_2(:,2), 'g.');
xtitle('LHS random','x1','x2');
