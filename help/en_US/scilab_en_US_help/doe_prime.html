doe_prime

Returns any of the first PRIME_MAX (1600) prime numbers.

Calling Sequence

p = doe_prime(n)

Parameters

n

the rank of the prime number to be returned

p

the nth stored prime number

Description

Returns any of the first PRIME_MAX (1600) prime numbers.

Examples

disp(doe_prime(10))
disp(doe_prime(1234))
