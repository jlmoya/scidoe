kurtosis

Compute the Kurtosis of a distribution of points.

Calling Sequence

retval = kurtosis(x)

Parameters

x

the sample of points on which we want to compute the Kurtosis

retval

the value of the Kurtosis for the considered set of points

Description

Compute the Kurtosis of a distribution of points.

Examples

A = 0.3*rand(100,1) + 1;
Res = kurtosis(A);
printf('The Kurtosis of the set A is %f\n', Res);
