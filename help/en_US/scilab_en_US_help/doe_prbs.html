doe_prbs

Returns a PRBS (Pseudo Random Binary signal) sequence

Calling Sequence

Result = doe_prbs(init,feedback)

Parameters

init

the initial value of the sequence (a vector of 0 or 1)

feedback

a vector of position indicating where are the feedbacks

Result

the sequence (you must call several times doe_prbs to get it evolves)

Description

Returns a PRBS (Pseudo Random Binary signal) sequence.

Examples

Res = doe_prbs(ones(1,10),[1 3 5]);
for i=1:20
  disp(doe_prbs(Res,[1 3 5]);
end
