crossvalidate

Perform K-fold cross validation on a function.

Calling Sequence

[s_opt,b_opt,res_mean,res_std] = crossvalidate(fun,K,steps,X,y,varargin)

Parameters

fun

the function on which we will perform the cross validation

K

the number of sets sampled

steps

the number of equidistant positions along fun at which the sum of squared residuals (SSR) is measured. Typically this is some large number to ensure sufficent accuracy.

X

the data matrix used as input to fun.

y

the response.

varargin

an arbitrary number of parameters sent to fun.

s_opt

0 < s_opt <= 1, determines the optimal model position.

b_opt

the optimal parameter.

res_mean

the mean SSR curve.

res_std

the standard deviations of the SSR curve.

Description

Performs simple K-fold cross validation on function fun.

Note: This is merely a simple implementation that has been tested with the lars function only. Use cautiously.

Examples

loadmatfile('diabetes.mat');

X = diabetes.x; size(X)
X = normalize(X);
y = diabetes.y;
y = center(y);
[n p] = size(X);

b1 = lars(X, y, 'lasso', 0, 0, [], 1);
s1 = sum(abs(b1),2)/sum(abs(b1(size(b1,1),:)));

plot(s1, b1, '-');
plot(s1, b1, 'ro');

xtitle('LARS','lambda','param');

[s_opt, b_opt, res_mean, res_std] = crossvalidate(lars, 10, 1000, X, y, 'lasso', 0, 0, [], 0);

cvplot(s_opt, res_mean, res_std);
