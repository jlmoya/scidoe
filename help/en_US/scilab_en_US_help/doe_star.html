doe_star

Produce a star design of experiments

Calling Sequence

H = doe_star(nb_var)

Parameters

nb_var

the number of variables in the design of experiments

H

the design of experiments

Description

Produce a star design of experiments

Examples

model_lin        = '1 x1 x2';
model_poly_inter = '1 x1 x2 x1*x2';
model_poly_2nd   = '1 x1 x2 x1*x2 x1^2 x2^2';

x = -1:0.1:1;
y = -1:0.1:1;
[X,Y] = meshgrid(x,y);

////////////////////////////
// Test of classicals DoE //
////////////////////////////

// Test of the star DoE
H2  = doe_star(2);
RM2 = build_regression_matrix(H2,model_lin);
