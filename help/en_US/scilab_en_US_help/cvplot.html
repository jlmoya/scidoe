cvplot

Simple plotting function for cross validation results.

Calling Sequence

cvplot(s_opt,res_mean,res_std)

Parameters

s_opt

0 < s_opt <= 1, determines the optimal model position.

res_mean

the mean SSR curve.

res_std

the standard deviations of the SSR curve.

Description

Simple plotting function for cross validation results.

cvplot(s_opt, res_mean, res_std) plots the mean reconstruction error with error bars resulting from the function CROSSVALIDATE. The optimal model choice is marked with a dashed red line.

Examples

loadmatfile('diabetes.mat');

X = diabetes.x; size(X)
X = normalize(X);
y = diabetes.y;
y = center(y);
[n p] = size(X);

b1 = lars(X, y, 'lasso', 0, 0, [], 1);
s1 = sum(abs(b1),2)/sum(abs(b1(size(b1,1),:)));

plot(s1, b1, '-');
plot(s1, b1, 'ro');

xtitle('LARS','lambda','param');

[s_opt, b_opt, res_mean, res_std] = crossvalidate(lars, 10, 1000, X, y, 'lasso', 0, 0, [], 0);

cvplot(s_opt, res_mean, res_std);
