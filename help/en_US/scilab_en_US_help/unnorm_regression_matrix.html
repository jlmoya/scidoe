unnorm_doe_matrix

Translate a design of experiments matrix from the -1 / 1 levels to the min / max levels.

Calling Sequence

H_unnorm = unnorm_doe_matrix(H,min_levels,max_levels)

Parameters

H

the regression matrix to be translated

min_levels

the values for each variables which will correspond to the -1 level.

max_levels

the values for each variables which will correspond to the 1 level.

H_unnorm

the translated design of experiments

Description

Translate a design of experiments matrix from the 0 / 1 levels to the min / max levels.

Examples

model_poly_inter = '1 x1 x2 x1*x2';

x = -1:0.1:1;
y = -1:0.1:1;
[X,Y] = meshgrid(x,y);

////////////////////////////
// Test of classicals DoE //
////////////////////////////

// Test of the factorial DoE
H1  = doe_factorial(2);

min_levels = [-1 -2];
max_levels = [2 3];

H1_unnorm = unnorm_doe_matrix(H1,min_levels,max_levels);

disp(H1)
disp(H1_unnorm)
