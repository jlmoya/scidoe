doe_test_var

This statistical test is based on the Fisher test. this test allows to test the variance of two samples

Calling Sequence

result = doe_test_var(A,B,level,operation)

Parameters

A

the first sample set to be compared

B

the second sample set to be compared

level

the level of confidence for the comparison

operation

which kind of operation do we want to perform. Can be '==', '>=' or '<='

result

a boolean corresponding to the result of the test.

Description

This statistical test is based on the Fisher test. this test allows to test the variance of two samples

Examples

rand('normal');
A = 0.1*rand(100,1) + 3;
B = 0.2*rand(100,1) + 3.1;
disp(doe_test_var(A,B,0.05,'=='));
disp(doe_test_var(A,B,0.05,'>='));
disp(doe_test_var(A,B,0.05,'<='));
