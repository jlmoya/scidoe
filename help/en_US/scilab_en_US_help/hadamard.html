hadamard

Build a Hadamard matrix.

Calling Sequence

H = hadamard(n)

Parameters

n

the number of time the elementary 2 x 2 Hadamard matrix is repeated

H

the Hadamard matrix (of size 2^n x 2^n)

Description

Build a Hadamard matrix.

Examples

disp(hadamard(4));
