normalize

Normalize the observations of a data matrix.

Calling Sequence

X_norm = normalize(X_in,replace)

Parameters

X_in

the data set to be normalized

replace

a boolean

X_norm

the normalized data set

Description

Normalize the observations of a data matrix.

X = normalize(X) centers and scales the observations of a data matrix such that each variable (column) has unit length.

Examples

A = 3*rand(30,3) + 1;
Y = normalize(A);
disp(max(Y,'r') - min(Y,'r'))
