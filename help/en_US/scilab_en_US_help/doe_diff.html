doe_diff

Compute the set of common points between two design of experiments

Calling Sequence

H = doe_diff(H1,H2)

Parameters

H1

a first set of design of experiments

H2

a second set of design of experiments

H

the common set of points between H1 and H2

Description

Compute the set of common points between two design of experiments.
