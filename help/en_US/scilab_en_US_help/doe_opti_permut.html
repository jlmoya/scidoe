doe_opti_permut

Apply the algorithm used in the doe_[gado]_opti function to a customized criterion.

Calling Sequence

[M_doe,history] = doe_opti_permut(M_init,M_cand,doe_size,model,criterion,l_bounds,u_bounds,ItMX,p_level,Log)

Parameters

M_init

an initial design of experiment

M_cand

a set of candidate points

doe_size

the desired number of points we want if the optimized design of experiments

model

a list of monomials like '1 x1 x2, x1*x2' where xi corresponds to the ith column

criterion

a function given by the user which returns the efficiency of the given design of experiment. The criterion function must have the following prototype: criterion(M_doe, model)

l_bounds

the vector of lower bounds for each variables

u_bounds

the vector of upper bounds for each variables

ItMX

the maximum number of iterations to perform

p_level

the progress level. Below this level of improvement between 2 iterations, the algorithm stops

Log

if %T then some intermediate messages are printed in the console

M_doe

the optimized design of experiments

history

the value of the given criterion, iterations after iterations

Description

Apply the algorithm used in the doe_[gado]_opti function to a customized criterion.

Examples

Doe_size   = 25;
Doe_ItMX   = 10;
Doe_plevel = 0.05;
Log        = %T;

model = doe_poly_model('inter',2,1);

Doe_Upper = ones(1,2);
Doe_Lower = -1*ones(1,2);

[X, Y] = meshgrid(-1:0.1:1, -1:0.1:1);
Index = 1;
for i=1:size(X,1)
  for j=1:size(X,2)
    Doe_Db(Index,:) = [X(i,j), Y(i,j)];
    Index = Index + 1;
  end
end

Doe_Init = [];

// Definition of the criterion function

function Result=comp_ssd_d_value_crit(M_doe, model)
  H      = build_regression_matrix(M_doe,model);
  n      = size(H,2);
  IMat   = H'*H / n;
  Result = -(prod(spec(IMat))^(1/n));
endfunction

scf;
drawlater;
if ~isempty(Doe_Init) then
  subplot(1,2,1);
  for i=1:size(Doe_Init,1)
    plot(Doe_Init(i,1), Doe_Init(i,2), 'ro');
  end
  xtitle('Initial DoE','x1','x2');
end
if ~isempty(Doe_Init) then
  subplot(1,2,2);
end
for i=1:size(Doe_Db,1)
  plot(Doe_Db(i,1), Doe_Db(i,2), 'ro');
end
xtitle('Points database','x1','x2');
drawnow;

[Doe_Comp, history] = doe_opti_permut(Doe_Init, Doe_Db, Doe_size, model, comp_ssd_d_value_crit, Doe_Lower, Doe_Upper, Doe_ItMX, Doe_plevel, Log);

RM = build_regression_matrix(Doe_Comp, model);

// Computation of the variance
[X, Y] = meshgrid(-1:0.1:1, -1:0.1:1);
for i=1:size(X,1)
  for j=1:size(X,2)
    x_aux = [X(i,j) Y(i,j)]';
    Z(i,j) = var_regression_matrix(RM, x_aux, model, 1);
  end
end

scf;
drawlater;

subplot(2,2,1)
surf(X,Y,Z);
xtitle('Custom D-Opti DoE','x1','x2', 'Var');

subplot(2,2,2)
for i=1:size(Doe_Comp,1)
  plot(Doe_Comp(i,1), Doe_Comp(i,2), 'ro');
end

subplot(2,1,2)
t=1:length(history);
plot(t, history, 'k');
xtitle('History of the criterion','Iterations','Criterion');
drawnow;
