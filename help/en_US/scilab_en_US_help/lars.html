lars

The LARS algorithm for performing LAR or LASSO.

Calling Sequence

beta = lars(X,y,method,stop,usegram,gram,trace)

Parameters

X

the input data set (a npoints x nvars matrix)

y

the output measures data set (a npoints x 1 matrix)

method

the method used (optional parameter). Can be:

  • 'lars': to select the least angle regression

  • 'lasso': to select the lasso regression

stop

the early stopping criterion (optional parameter). If stop is positive, stop corresponds to an upper bound on the L1-norm of the beta coefficients. If stop is negative, stop is an integer that determines the desired number of variables

usegram

specifies whether the Gram matrix X'X should be calculated (usegram = 1) or not (usegram = 0) (optional parameter). Calculation of the Gram matrix is suitable for low-dimensional problems. By default, the Gram matrix is calculated.

gram

makes it possible to supply a pre-computed Gram matrix (optional parameter). Set usegram to 1 to enable. If no Gram matrix is available, exclude argument or set gram = [].

trace

with nonzero trace will print the adding and subtracting of variables as all lars / lasso solutions are found (optional parameter).

beta

each row of beta contains the predictor coefficients of one iteration. A suitable row is chosen using e.g. cross-validation, possibly including interpolation to achieve sub-iteration accuracy.

Description

beta = lars(X, Y) performs least angle regression on the variables in X to approximate the response Y.

Variables X are assumed to be normalized (zero mean, unit length), the response Y is assumed to be centered.

Examples

loadmatfile('diabetes.mat');

X = diabetes.x; size(X)
X = normalize(X);
y = diabetes.y;
y = center(y);
[n p] = size(X);

b1 = lars(X, y, 'lasso', 0, 0, [], 1);
s1 = sum(abs(b1),2)/sum(abs(b1(size(b1,1),:)));

plot(s1, b1, '-');
plot(s1, b1, 'ro');

xtitle('LARS','lambda','param');

[s_opt, b_opt, res_mean, res_std] = crossvalidate(lars, 10, 1000, X, y, 'lasso', 0, 0, [], 0);

cvplot(s_opt, res_mean, res_std);
